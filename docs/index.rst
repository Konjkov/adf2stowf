ADF2STOWF Documentation

About ADF2STOWF
===============

ADF2STOWF provides utilities for reading ADF output files and converting them to various formats for further quantum chemical analysis. The library includes functionality for handling molecular orbitals, basis sets, and other quantum chemical data.
